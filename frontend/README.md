## Getting started

Clone from GitHub

```bash
git clone https://github.com/puddingii/MyLittleAccountBook.git
cd frontend
yarn
yarn start
```

## Technology stack

- [Material UI V5](https://mui.com/core/)
- Built with React Hooks API
- Recoil & React Query
- React Router for navigation routing
- Support for react-script
- Code splitting
- CSS-in-JS
